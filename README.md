# 038 Transaction

[![038 Transaction](https://img.youtube.com/vi/ABVdxhYq768/0.jpg)](https://www.youtube.com/watch?v=ABVdxhYq768)

The SQL in the `exercise.sql` file
- Should create a transaction that succesfully completes the following
    - Inserts a row into the `Stores` table with an `ID` of "1", and a not null value in each of the other columns
    - Inserts a row into the `Employee` table with an `ID` of "1050",  and a not null value in each of the other columns
    - Inserts a row into the `Contact_Info` table with an `ID` of "1050", and a not null value in each of the other columns
    - Change the `NEXT_EXPIRE_DATE` to today's date for "Spinach", "Avocado", and "Yogurt" in the `Grocery` table
